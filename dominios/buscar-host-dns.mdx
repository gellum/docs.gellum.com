Esta documentación te guiará a través del proceso de utilizar **Whois** para descubrir el host DNS de un dominio, proporcionándote los pasos claros y sencillos para que puedas realizar esta búsqueda de manera efectiva.

En la mayoría de los casos puedes utilizar herramientas de línea de comandos si tienes conocimientos técnicos más avanzados (como el comando whois en sistemas Linux o macOS)

```bash
whois gellum.com
```

También puedes usar herramientas como grep para filtrar la salida del comando whois y mostrar sólo las lineas que contienen la información de los servidores de nombres (NS).

```bash
whois google.com | grep "Name Server"
```

o, una alternativa similar:

```bash
whois google.com | grep "Nserver"
```

Esperamos que esta explicación sea de ayuda para que pueda verificar el host DNS de un dominio.

<Tip>
  Sabiendo el host DNS podrá modificar sus registros DNS para [configurar los registros MX de Gellum Mailbox](/email/registros) y otros registros importantes.
</Tip>
