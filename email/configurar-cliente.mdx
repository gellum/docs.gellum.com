## Configuración predeterminada del servidor

Use IMAP o POP para configurar nuestros servidores. Estos proporcionan una conexión segura mediante uso de SSL/TLS

| ### Protocolo   | ### Servidor seguro (SSL) | ### Número de Puerto (SSL) |
| --------------- | ------------------------- | -------------------------- |
| IMAP (entrante) | secure.emailsrvr.com      | 993 (SSL)                  |
| POP3 (entrante) | secure.emailsrvr.com      | 995 (SSL)                  |
| SMTP (saliente) | secure.emailsrvr.com      | 465 (SSL)                  |

Además necesitará usar su usario y contraseña en los siguientes campos:

| Nombre de usuario               | Contraseña                 |
| ------------------------------- | -------------------------- |
| Tu dirección de correo completa | La contraseña de tu correo |
